git使用

发表于 2018-01-30

pwd 命令是用于显示当前的目录。

git init 把这个目录变成git可以管理的仓库

git add readme.txt 添加到暂存区里面去

git commit -m “注释” 把文件提交到本地仓库

git status 来查看是否还有文件未提交

git diff readme.txt 文件到底改了什么内容

git log 最近三次提交历史记录

git log –pretty=oneline 最近三次提交历史记录

23种设计模式

发表于 2018-01-17

三大类：

创建型模式，共五种：工厂方法模式、抽象工厂模式、单例模式、建造者模式、原型模式。

1、工厂方法模式（Factory Method）

总结：定一个工厂类，内部生成产品对象。

工厂方式模式分为三种：

11、普通工厂模式

22、多个工厂方法模式

33、静态工厂方法模式

总体来说，工厂模式适合：凡是出现了大量的产品需要创建，并且具有共同的接口时，可以通过工厂方法模式进行创建。在以上的三种模式中，第一种如果传入的字符串有误，不能正确创建对象，第三种相对于第二种，不需要实体化工厂类，所以，大多数情况下，我们会选用第三种–静态工厂方法模式。
